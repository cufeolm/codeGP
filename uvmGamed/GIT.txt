git status === review changed files - unstaged (red) - staged (green)
git add .
git commit -m "message"
git push

git log === reviews history of commits - to close history press 'q'
git diff === to review local changes



git fetch === fetch all remote branches

git checkout -b branch-name === create new local branch

git checkout branch-name === switch to another local branch

git branch -d branch-name === delete branch locally

git pull origin master === merge master remote changes to your current local branch

git branch === current local branch

git stash === save local changes

git stash list === list saved changes

git stash apply === apply saved changes
